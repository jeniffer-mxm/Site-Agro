==========================================
🚀 COMO ABRIR O PROJETO NO NAVEGADOR:
==========================================

1. Abra o arquivo "index.html" diretamente no navegador
2. Ou use um servidor local (recomendado):
   - Python: python -m http.server 8000
   - Node.js: npx http-server
   - VS Code: Live Server extension

==========================================
🖼️ ONDE SUBSTITUIR IMAGENS:
==========================================

🔹 LOGO DA EMPRESA:
   - Local: /img/logo.png
   - Formato: PNG com fundo transparente
   - Tamanho recomendado: 48x48px ou 96x96px
   - O site detectará automaticamente quando a logo for inserida

🔹 IMAGENS DOS PRODUTOS:
   - Tratores: /img/trator-placeholder.jpg
   - Máquinas: /img/maquina-placeholder.jpg  
   - Jatinhos: /img/jatinho-placeholder.jpg
   - Formato: JPG ou PNG
   - Tamanho recomendado: 800x600px

🔹 IMAGEM DE FUNDO DO HERO:
   - Atualmente usando URL externa (Unsplash)
   - Para usar imagem própria: substitua a URL no CSS
   - Arquivo: css/style.css (linha ~200 aproximadamente)
   - Procure por: background-image

==========================================
📝 COMO EDITAR TEXTOS E CORES:
==========================================

🔹 TEXTOS:
   - Todos os textos estão no arquivo: index.html
   - Procure pelas seções: #home, #sobre, #produtos, #servicos, #contato
   - Edite diretamente o conteúdo HTML

🔹 CORES (se necessário alterar):
   - Arquivo: css/style.css
   - Procure por ":root" no início do arquivo
   - Variáveis disponíveis:
     * --bg-primary: Cor do fundo principal
     * --accent-primary: Cor verde agro (#228B22)
     * --text-primary: Cor do texto principal
   
🔹 NOME DA EMPRESA:
   - Procure por "Agro" no index.html
   - Substitua por seu nome desejado
